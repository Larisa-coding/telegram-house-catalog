# Прямое подключение репозитория на Railway

## ✅ САМЫЙ ПРОСТОЙ СПОСОБ - Railway Deploy Button

Я добавил кнопку деплоя в README.md. Просто:

1. **Откройте репозиторий на GitHub**: https://github.com/Larisa-coding/telegram-house-catalog
2. **Нажмите на кнопку "Deploy on Railway"** в README
3. Railway автоматически:
   - Создаст проект
   - Подключит репозиторий
   - Начнет деплой

## Альтернатива: Прямая ссылка

Если кнопка не работает, используйте прямую ссылку:

**Откройте в браузере:**
```
https://railway.app/template/new?repository=https://github.com/Larisa-coding/telegram-house-catalog
```

Railway автоматически создаст проект и подключит репозиторий!

## Если все еще не работает

1. **Откройте**: https://railway.app/new
2. **Выберите**: "Deploy from GitHub repo"
3. **В поле поиска вставьте**: `Larisa-coding/telegram-house-catalog`
4. **Или полный URL**: `https://github.com/Larisa-coding/telegram-house-catalog.git`

---

**Важно**: После подключения репозитория Railway автоматически определит Node.js проект и начнет деплой.
