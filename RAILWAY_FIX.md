# Решение: Репозиторий не отображается в Railway

## Быстрое решение

### Вариант 1: Обновить список репозиториев

1. **Нажмите на "Configure GitHub App"** (с иконкой шестеренки)
2. Или нажмите на иконку обновления (refresh) справа
3. Подождите несколько секунд
4. Список должен обновиться и показать `telegram-house-catalog`

### Вариант 2: Переподключить GitHub

1. Нажмите "Configure GitHub App"
2. Выберите "Reconnect" или "Disconnect and reconnect"
3. Авторизуйтесь заново через GitHub
4. Разрешите доступ к репозиториям
5. Проверьте список снова

### Вариант 3: Создать проект вручную и подключить репозиторий

Если репозиторий все еще не появляется:

1. **Создайте пустой проект**:
   - Railway → "New Project" → "Empty Project"

2. **Добавьте сервис**:
   - В проекте нажмите "+ New" → "GitHub Repo"
   - В поле поиска введите: `telegram-house-catalog`
   - Или вставьте полный путь: `Larisa-coding/telegram-house-catalog`

3. **Или используйте Deploy Button**:
   - Railway автоматически определит репозиторий

### Вариант 4: Проверить репозиторий на GitHub

Убедитесь, что:
- Репозиторий существует: https://github.com/Larisa-coding/telegram-house-catalog
- Репозиторий не в архиве
- Вы авторизованы в Railway под правильным аккаунтом (Larisa-coding)

### Вариант 5: Подождать синхронизацию

Если репозиторий только что создан:
- Подождите 1-2 минуты
- Обновите страницу Railway
- Нажмите "Configure GitHub App" → "Refresh"

### Вариант 6: Использовать Railway CLI

```bash
# Установите Railway CLI
npm install -g @railway/cli

# Авторизуйтесь
railway login

# Создайте проект
railway init

# Подключите репозиторий
railway link
```

---

**Рекомендация**: Попробуйте сначала Вариант 1 (обновить список), затем Вариант 3 (создать проект вручную).
